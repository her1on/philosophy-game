# Философия — CLAUDE.md

Контекст для новых сессий. Читай этот файл перед тем как что-то менять.

## Что это

Образовательная игра по истории западной философии. Командный проект — каждый участник делает свою локацию (эпоху). Сделана одна локация — «Новейшая западная философия».

**Два файла в активной разработке:**
- **`index.html`** — главное меню со шкалой времени и карточками эпох
- **`philosophia.html`** — локация «Новейшая западная философия» (газетный стиль)

Сайт: **https://philosophy-game.onrender.com**  
Репо: **https://github.com/her1on/philosophy-game**

## Структура проекта (командная)

Каждый участник делает свою локацию — отдельный HTML-файл. Готовая локация добавляется в массив `EPOCHS` в `index.html` с `available: true` и `url: 'имя_файла.html'`.

```
index.html          ← общее меню (один человек)
philosophia.html    ← Новейшая западная (сделано)
location2.html      ← Античность (чужая локация, когда будет готова)
location3.html      ← Средневековье (чужая локация)
...
```

## Главное меню (index.html)

### Визуальный стиль
- **Концепция:** тёмный кабинет философа — атмосфера старой библиотеки, свет из окна
- Фон: `#0e0b06` + тёплый янтарный radial-gradient сверху
- Типографика: Georgia, serif; золото `#c8a84b`; text-shadow glow
- Геометрический орнамент в углу (через CSS border-radius + background-image)
- Шкала времени: золотая линия, тёмные точки, активная эпоха светится
- Карточки эпох: пергаментный фон `#e8d9b0→#d4c490`, лёгкие повороты через `nth-child`, тени на тёмном фоне

### Как добавить новую локацию в меню

Найти массив `EPOCHS` в `index.html` и добавить/изменить объект:

```js
{
  id: 'ancient',
  name: 'Античность',
  period: 'VI в. до н.э. — V в. н.э.',
  year: 'VI в. до н.э.',
  desc: 'Краткое описание для карточки.',
  available: true,          // ← было false, стало true
  url: 'ancient.html',      // ← файл локации
}
```

## Архитектура игры (philosophia.html)

### Навигация — 3 уровня

```
Карта города (screen-map)
  └── 2 лагеря — маркеры на карте (screen-camp)
        ├── Антисциентистские направления (3 направления)
        └── Сциентистские направления (6 направлений)
              └── Встреча с направлением (screen-encounter)
                    └── Квиз (screen-quiz)
                          └── Финал (screen-done)
```

Прогресс хранится в `localStorage` под ключом `philo_v2`.

### Визуальный стиль
- **Концепция:** старая газета / «Философский Вестник», XIX век
- Фон: `#e8dfc0` с бумажной сеткой через `repeating-linear-gradient`
- Рамки: `#9a7040` (коричневый)
- Шрифт: Georgia, serif
- Кнопки: `#2a4a20` (тёмно-зелёный) / `rgba(255,252,240)`
- Баннер направления: градиент `#c8b060 → #d4c080 → #c0a850`
- Дореформенная орфография намеренна (стиль XIX века): «ВѢРНО», «ГОВОРИТЪ»

### Карта (экран выбора лагеря)
- **Фон:** `img/map.jpeg` — рисованная антикварная карта двух территорий:
  тёмный дикий лес/горы слева (антисциентисты), светлый город науки справа
  (сциентисты), река с мостом посередине. Размер 1792×1024 (соотношение 1.75).
- **SVG-оверлей** поверх картинки: `viewBox="0 0 896 512"` (ровно 1.75, маппинг
  1:1 с картинкой, без зазоров). Если меняешь картинку — подгони `viewBox` под её
  пропорции, иначе метки «съедут».
- **Метки лагерей** рисуются в JS (`renderMarkers`): печать-медальон + лента с
  подписью (`camp.short`) + счётчик «пройдено X/N». Позиция — поля `mx/my` в
  координатах viewBox 896×512. Сейчас: антисциентисты `152,205` (деревня в лесу),
  сциентисты `512,158` (купол обсерватории).
- Цвет печати по лагерю: антисциентисты `#8b2e1a` (багровый), сциентисты `#244b63`
  (синий); пройденный лагерь — зелёный `#2f6a26` с галочкой.
- Компас и пунктирный маршрут были, но **удалены** (не легли на рисунок).
  Тонировка-градиент зон тоже убрана — цвета территорий уже в самой картинке.

### Контрастные темы лагерей (экран направлений)
Экран `#screen-camp` получает класс `theme-anti` / `theme-sci` (в `enterCamp`):
- **theme-anti** — тёмный драматичный фон, багровый акцент, надзаголовок
  «Лагерь I · по ту сторону разума».
- **theme-sci** — светлый «чертёжный» фон с синей сеткой, надзаголовок
  «Лагерь II · во имя науки».

### Переходы (анимации)
- `.screen` — плавное появление (fade + сдвиг) при показе экрана.
- Метки лагерей — `markerPop` (отскок) со stagger.
- Карточки направлений — каскад через класс `.in` + `transition-delay`.
- Баннер направления — `unfurl` (разворот сверху).

## Формат данных направления (philosophia.html)

```js
{
  id: 'life',                        // уникальный id, латиница
  name: 'Философия жизни',
  philosophers: 'А. Шопенгауэр  ·  Ф. Ницше  ·  А. Бергсон',  // через ·
  desc: 'Описание, 3–5 предложений.',
  quote: '«Цитата.»',                // без имени автора
  quoteCite: '— Ф. Ницше',          // имя автора отдельно
  tags: ['Тег 1', 'Тег 2'],          // 4–7 тегов
  hint: 'Подсказка (резерв, сейчас в квизе не используется).',
  quiz: [                              // ← массив вопросов (обычно 2)
    {
      type: 'choice',                 // choice | who | gap | truefalse
      q: 'Вопрос?',
      opts: ['А', 'Б', 'В', 'Г'],     // варианты (truefalse — по умолчанию Вѣрно/Невѣрно)
      ans: 2,                         // индекс правильного (0–3)
      why: ['…', '…', '✓ Верно: …', '…']  // объяснение к КАЖДОМУ варианту
    },
    {
      type: 'who',                    // who/gap — можно дать цитату:
      quote: '«Цитата с ___».',       // у gap — с пропуском «___»
      q: 'Чьи это слова? / вставьте слово',
      opts: ['…', '…', '…', '…'],
      ans: 1,
      why: ['…', '✓ Верно: …', '…', '…']
    }
  ],
  successQuote: '«Цитата при завершении направления.»',
  successCite: '— Автор'
}
```

**Типы вопросов** (все — «выбери один вариант», движок общий):
- `choice` — обычный выбор из 4.
- `who` — «кто это сказал?» (опц. поле `quote` показывается крупно).
- `gap` — пропуск в цитате (`quote` с `___`, варианты — слова).
- `truefalse` — утверждение; `opts` можно не задавать (будет Вѣрно/Невѣрно).

После ответа показывается разбор всех вариантов (`why[i]`), затем «Дальше →»
или «Записать в блокнот →» на последнем вопросе. Направление отмечается пройденным
по завершении всех вопросов (подход обучающий, без штрафа за ошибку).
