# Философия — CLAUDE.md

Контекст для новых сессий. Читай этот файл перед тем как что-то менять.

## Что это

Образовательная игра по истории западной философии. Командный проект — каждый участник делает свою локацию (эпоху). Сделана одна локация — «Новейшая западная философия».

**Два файла в активной разработке:**
- **`index.html`** — главное меню со шкалой времени и карточками эпох
- **`philosophia.html`** — локация «Новейшая западная философия» (газетный стиль)

Сайт: **https://philosophy-game.onrender.com**  
Репо: **https://github.com/her1on/philosophy-game**

## Структура проекта (командная)

Каждый участник делает свою локацию — отдельный HTML-файл. Готовая локация добавляется в массив `EPOCHS` в `index.html` с `available: true` и `url: 'имя_файла.html'`.

```
index.html          ← общее меню (один человек)
philosophia.html    ← Новейшая западная (сделано)
location2.html      ← Античность (чужая локация, когда будет готова)
location3.html      ← Средневековье (чужая локация)
...
```

## Главное меню (index.html)

### Визуальный стиль
- **Концепция:** тёмный кабинет философа — атмосфера старой библиотеки, свет из окна
- Фон: `#0e0b06` + тёплый янтарный radial-gradient сверху
- Типографика: Georgia, serif; золото `#c8a84b`; text-shadow glow
- Геометрический орнамент в углу (через CSS border-radius + background-image)
- Шкала времени: золотая линия, тёмные точки, активная эпоха светится
- Карточки эпох: пергаментный фон `#e8d9b0→#d4c490`, лёгкие повороты через `nth-child`, тени на тёмном фоне

### Как добавить новую локацию в меню

Найти массив `EPOCHS` в `index.html` и добавить/изменить объект:

```js
{
  id: 'ancient',
  name: 'Античность',
  period: 'VI в. до н.э. — V в. н.э.',
  year: 'VI в. до н.э.',
  desc: 'Краткое описание для карточки.',
  available: true,          // ← было false, стало true
  url: 'ancient.html',      // ← файл локации
}
```

## Архитектура игры (philosophia.html)

### Навигация — 3 уровня

```
Карта города (screen-map)
  └── 2 лагеря — маркеры на карте (screen-camp)
        ├── Антисциентистские направления (3 направления)
        └── Сциентистские направления (6 направлений)
              └── Встреча с направлением (screen-encounter)
                    └── Квиз (screen-quiz)
                          └── Финал (screen-done)
```

Прогресс хранится в `localStorage` под ключом `philo_v2`.

### Визуальный стиль
- **Концепция:** старая газета / «Философский Вестник», XIX век
- Фон: `#e8dfc0` с бумажной сеткой через `repeating-linear-gradient`
- Рамки: `#9a7040` (коричневый)
- Шрифт: Georgia, serif
- Кнопки: `#2a4a20` (тёмно-зелёный) / `rgba(255,252,240)`
- Баннер направления: градиент `#c8b060 → #d4c080 → #c0a850`
- Дореформенная орфография намеренна (стиль XIX века): «ВѢРНО», «ГОВОРИТЪ»

## Формат данных направления (philosophia.html)

```js
{
  id: 'life',                        // уникальный id, латиница
  name: 'Философия жизни',
  philosophers: 'А. Шопенгауэр  ·  Ф. Ницше  ·  А. Бергсон',  // через ·
  desc: 'Описание, 3–5 предложений.',
  quote: '«Цитата.»',                // без имени автора
  quoteCite: '— Ф. Ницше',          // имя автора отдельно
  tags: ['Тег 1', 'Тег 2'],          // 4–7 тегов
  hint: 'Подсказка для квиза.',
  quiz: {
    q: 'Вопрос?',
    opts: ['А', 'Б', 'В', 'Г'],      // всегда 4 варианта
    ans: 2                            // индекс (0–3)
  },
  successQuote: '«Цитата при правильном ответе.»',
  successCite: '— Автор'
}
```

## Важные решения

- **Все данные внутри файлов** — никаких внешних JSON, `philosophia.html` самодостаточен
- **localStorage ключ `philo_v2`** — смени ключ если меняешь структуру данных (сбросит прогресс)
- **Карта фиксированная** — `img/map.jpeg` 860×460, маркеры лагерей позиционируются через `mx/my`
- **Квиз: всегда 4 варианта**, правильный — поле `ans` (индекс 0–3)
- **3D удалён** — Three.js, A-Frame, engine.js, все minigame-файлы удалены в коммите `c322441`
