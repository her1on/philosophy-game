## Что это

Образовательная RPG-игра по истории западной философии. Командный проект — каждый участник делает свою локацию (эпоху). Сделана одна локация — «Новейшая западная философия».

**Два файла в активной разработке:**
- **`index.html`** — главное меню со шкалой времени и карточками эпох
- **`philosophia.html`** — локация «Новейшая западная философия», RPG-ходилка

Сайт: **https://philosophy-game.onrender.com**
Репо: **https://github.com/her1on/philosophy-game**

## Структура проекта (командная)

Каждый участник делает свою локацию — отдельный HTML-файл. Готовая локация добавляется в массив `EPOCHS` в `index.html` с `available: true` и `url: 'имя_файла.html'`.

```
index.html          ← общее меню (один человек)
philosophia.html    ← Новейшая западная (сделано)
ancient.html        ← Античность (чужая локация, когда будет готова)
...
```

### Как добавить новую локацию в меню

```js
// В массиве EPOCHS в index.html:
{
  id: 'ancient',
  name: 'Античность',
  period: 'VI в. до н.э. — V в. н.э.',
  year: 'VI в. до н.э.',
  desc: 'Краткое описание.',
  available: true,        // ← было false
  url: 'ancient.html',   // ← файл локации
}
```

## Архитектура игры (philosophia.html)

### Навигация

```
Меню (index.html)
  └── Ходилка (screen-explore) — главный и единственный стартовый экран
        ├── подойти к философу (E) → Встреча (screen-encounter)
        │                               └── Квиз (screen-quiz)
        │                                     └── Финал (screen-done)
        └── оверлей «Направления» (кнопка в углу) — быстрый список 9 школ + прогресс
```

- Кнопка «←» на screen-encounter/quiz → возврат в ходилку (`startExplore()`)
- Выход из локации → `exitExplore()` → `location.href='index.html'`
- Прогресс хранится в `localStorage` под ключом `philo_v2`
- **screen-map и screen-camp удалены** вместе с их CSS и мёртвыми функциями (`showMap`, `renderMarkers`, `renderJournal`, `enterCamp`) — код вычищен.

## Режим прогулки (screen-explore)

### Карта

- **Файл:** `img/explore_map.png` (1536×1024)
- **Содержание:** пиксельный арт — тёмный лес и горы слева (антисциентисты), каменный город с обсерваторией справа (сциентисты), река с деревянным мостом по центру
- **Константы карты:** `EXP.W = 1536`, `EXP.H = 1024`

### Автомасштаб

Карта растягивается на весь экран без чёрных полос:

```js
EXP.scale = Math.max(w / EXP.W, h / EXP.H);
```

Работает на любом мониторе (MacBook, университетский 1920px и т.д.).

### Объект EXP

```js
const EXP = {
  running:false, raf:0, ctx:null, canvas:null, img:null, imgReady:false,
  W:1536, H:1024, cw:0, ch:0, zoom:1, camX:0, camY:0,
  scale:1, keys:{}, joy:{active:false, dx:0, dy:0},
  player:{ x:0, y:0, r:13, dir:'down', moving:false }, sprite:null,
  npcs:[], near:null, started:false,
  coll:null, collW:0, collH:0, collScale:0.5, collOK:false
};
```

### Управление

- **WASD** — движение (по `e.code`, не `e.key` — работает на русской раскладке)
- **E** (`KeyE`) — взаимодействие с NPC
- Тач/джойстик: виртуальный стик для мобильных

### Спрайт игрока

Файл `img/sprites/player.png` (832×1344), LPC-формат, кадры 64×64:

| Направление | Строка спрайта |
|-------------|----------------|
| вверх       | 8              |
| влево       | 9              |
| вниз        | 10             |
| вправо      | 11             |

8 кадров анимации ходьбы, кадр 0 — стояние.

### Коллизии

- **Файл:** `img/explore_mask.png`
- Белый пиксель = можно ходить, чёрный = стена
- Генерируется скриптом `scripts/gen_explore_mask.py` из файла с красными линиями дорожек
- Чтобы изменить проходимость: перерисуй красные линии в draw.chat → экспортируй → запусти скрипт

### NPC (философы на карте)

Координаты в долях карты (0..1), поле `NPC_FRAC`:

```js
const NPC_FRAC = {
  life:   [0.182, 0.185], psycho: [0.074, 0.414], exist:  [0.290, 0.349],
  phenom: [0.632, 0.132], pragma: [0.705, 0.805], posit:  [0.762, 0.178],
  analyt: [0.764, 0.514], herm:   [0.852, 0.641], postmod:[0.884, 0.808]
};
```

Маркеры — плавающие пины с портретом, бейдж с именем при приближении, зелёная галочка если пройдено. Анимация — боббинг через `Math.sin`.

Радиус триггера: `0.075 * EXP.W` (в пикселях карты).

Спавн игрока: на мосту (~центр карты), чтобы сразу можно было выбрать направление.

## Квиз

За сессию показываются **3 случайных вопроса** из пула (Fisher-Yates shuffle + slice).

**Типы вопросов:**
- `choice` — выбор из 4 вариантов
- `who` — «кто это сказал?» (опц. поле `quote` показывается крупно)
- `gap` — пропуск в цитате (`quote` с `___`)
- `truefalse` — верно/неверно

После ответа показывается разбор всех вариантов (`why[i]`). Без штрафа за ошибку — обучающий режим.

### Формат данных направления

```js
{
  id: 'life',
  name: 'Философия жизни',
  philosophers: 'А. Шопенгауэр  ·  Ф. Ницше  ·  Э. фон Гартман',
  cards: [
    { id:'nietzsche', name:'Ф. Ницше', years:'1844–1900', note:'...' }
    // портрет: img/portraits/<id>.jpg
  ],
  desc: 'Описание, 3–5 предложений.',
  quote: '«Цитата.»',
  quoteCite: '— Ф. Ницше',
  tags: ['Тег 1', 'Тег 2'],
  quiz: [
    {
      type: 'choice',
      q: 'Вопрос?',
      opts: ['А', 'Б', 'В', 'Г'],
      ans: 2,                        // индекс правильного (0–3)
      why: ['…', '…', '✓ Верно: …', '…']
    }
  ],
  successQuote: '«Цитата при завершении.»',
  successCite: '— Автор'
}
```

## Важные решения

- **Все данные внутри файлов** — никаких внешних JSON, `philosophia.html` самодостаточен
- **localStorage ключ `philo_v2`** — смени ключ если меняешь структуру данных (сбросит прогресс)
- **Управление по `e.code`** — не `e.key`, иначе не работает на русской раскладке
- **Автомасштаб** — `Math.max(w/W, h/H)`, не `Math.min` — карта заполняет весь экран
- **Радиус триггера NPC** — `0.075 * EXP.W`; меньше — триггер не срабатывает у зданий
- **Render иногда не автодеплоит** — если пуш есть а сайт не обновился, жми Manual Deploy
- **3D удалён** — Three.js, A-Frame и все minigame-файлы удалены в коммите `c322441`
