# Философский Детектив — CLAUDE.md

Контекст для новых сессий. Читай этот файл перед тем как что-то менять.

## Что это

Образовательная игра по истории западной философии. Два режима:

- **3D-детектив** (`index.html`) — Three.js сцена, тёмный космический стиль, навигация кликами по объектам
- **Газетный вестник** (`philosophia.html`) — плоский HTML, стиль старой газеты (бежевый фон, шрифт Georgia), **основной рабочий файл**

Сайт: **https://philosophy-game.onrender.com** (открывает philosophia.html напрямую)  
Репо: **https://github.com/her1on/philosophy-game**

## Архитектура игры (philosophia.html)

Три уровня навигации:

```
Карта города
  └── 2 лагеря (маркеры на карте)
        ├── Антисциентистские направления (3 направления)
        └── Сциентистские направления (6 направлений)
              └── Встреча с направлением
                    └── Вопрос (квиз)
```

Экраны: `screen-map` → `screen-camp` → `screen-encounter` → `screen-quiz` → `screen-done`

Прогресс хранится в `localStorage` под ключом `philo_v2`.

## Формат данных направления

Каждое направление в `philosophia.html` — объект внутри массива `directions` лагеря:

```js
{
  id: 'life',                        // уникальный id, латиница
  name: 'Философия жизни',           // название направления
  philosophers: 'А. Шопенгауэр  ·  Ф. Ницше  ·  А. Бергсон',  // строка через ·
  desc: 'Полное описание направления, 3–5 предложений...',
  quote: '«Цитата одного из философов.»',   // без имени автора
  quoteCite: '— Ф. Ницше',                  // имя автора отдельно
  tags: ['Воля к власти', 'Сверхчеловек', 'Иррационализм'],  // 4–7 тегов
  hint: 'Краткая подсказка для квиза (1–2 предложения).',
  quiz: {
    q: 'Вопрос с одним правильным ответом?',
    opts: ['Вариант А', 'Вариант Б', 'Вариант В', 'Вариант Г'],  // всегда 4
    ans: 2   // индекс правильного ответа (0–3)
  },
  successQuote: '«Цитата, которая показывается при правильном ответе.»',
  successCite: '— Автор'
}
```

## Формат данных для 3D-этапа (stages/*.js)

Используется движком `engine.js` для 3D-сцены. Структура отличается от `philosophia.html`:

```js
PhiloGame.registerStage({
  id: 'modern', name: 'Название эпохи', era: 'XIX — XX вв.',
  color: 0x7030a0, fog: 0x080015,
  intro: [ { speaker: '...', icon: '◈', text: '...' } ],  // диалог при входе
  sublocs: [
    {
      id: 'anti', name: 'Суб-локация', subtitle: '...', color: 0x9030c0, fog: 0x0d001e,
      directions: [
        {
          name: '...', color: 0xb050ff,
          philosophers: '...', desc: '...', quote: '...', tags: [...],
          dialogue: [ { speaker: '...', icon: '...', text: '...' } ]
          // нет quiz — квиз только в philosophia.html
        }
      ]
    }
  ]
});
```

## Как добавить новую локацию в 3D-режим

1. Скопировать `stages/TEMPLATE.js` → `stages/stage_ancient.js`
2. Заполнить данные (только текст, никакого Three.js)
3. Добавить в `index.html` перед `PhiloGame.start()`:
   ```html
   <script src="stages/stage_ancient.js"></script>
   ```
4. Локация появится в главном меню автоматически

## Важные решения

- **philosophia.html самодостаточен** — все данные внутри файла, никаких внешних JSON
- **localStorage ключ `philo_v2`** — если поменять структуру данных, нужно сменить ключ чтобы сбросить старый прогресс у пользователей
- **Карта фиксированная** — `img/map.jpeg` (860×460), маркеры позиционируются через `mx/my` в данных лагеря
- **Квиз всегда 4 варианта** — индексы 0–3, правильный задаётся полем `ans`
